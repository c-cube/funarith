Module Funarith__Int

Interface for Integers

This abstracts over a type and operation for integers.

A possible implementation for arbitrary precision numbers is Zarith, with module Z

module type FULL = Funarith.Int_intf.FULL
module type BASE = Funarith.Int_intf.BASE
module type DERIVED = Funarith.Int_intf.DERIVED
module Default : FULL with type t = int

Fixed precision

module Derive : functor (T : BASE) -> DERIVED with type t = T.t
